🚀 point d'entrée public/index.php

<?php

// Chargement autoloader Composer
require_once __DIR__ . '/../vendor/autoload.php';

// Chargement variables environnement
use Dotenv\Dotenv;
$dotenv = Dotenv::createImmutable(__DIR__ . '/..');
$dotenv->load();

// Configuration erreurs
if ($_ENV['APP_DEBUG'] === 'true') {
error_reporting(E_ALL);
ini_set('display_errors', 1);
}

// Routage simple
use App\Config\Router;
use App\Controllers\HomeController;

$router = new Router();
$router->get('/', [HomeController::class, 'index']);
$router->run();

💡 bonnes pratiques php

  • psr-4 - Autoloading avec namespace
  • composer - Gestion dépendances obligatoire
  • .env - Variables environnement sécurisées
  • mvc - Séparation logique/présentation
  • tests - Tests unitaires avec phpunit
  • logs - Logging avec monolog
  • sécurité - Validation/sanitisation données
  • cache - Optimisation performances

⚠️ sécurité importante

  • Jamais de code PHP dans public/
  • Fichier .env dans .gitignore
  • Validation toutes les entrées utilisateur
  • Échappement données avant affichage
  • Requêtes préparées pour BDD
  • HTTPS en production obligatoire
